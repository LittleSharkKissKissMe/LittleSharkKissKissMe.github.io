    （基本与或非门RS触发器相反）

4.触发器的状态

    0态：Q为0，Q非为1

    1态：Q为1，Q非为0

    不正常态：Q和Q非都为0或1（不互补）

    现态：触发器接受信号之前所处状态，符号为Q^n

    次态：触发器在接受信号之后建立的新的稳定状态，符号为Q^(n+1)

    真值表

        图片

        （现态和次态主要是用于保持状态时的区分）

        由卡诺图，可得

            Q^(n+1) = S + R非Q^n (特性方程)

            SR = 0 (约束条件)

            特别注意，或非门基本触发器和与非门基本触发器的特性方程完全一样，这是因为它们定义的输入端R和S不同，换句话说，特性方程相同是输入端更改名称的结果

D触发器

为了解决R和S不能同时为0或1，对触发器进行改进

图片

输入仅有一个D，通过非门分化为两个输入，D和非D。其他基本与同步触发器一致。

S=D,R=D非，代入特性方程，得

    Q^(n+1) = D (**输出与Q^n无关**)

同步JK触发器

图片

输入端为J和K，分化为S = JQ^n非, R = K非Q^n

代入特性方程：

    Q^(n+1) = JQ^n非 + K非Q^n

图片

将J和K用T相连。T触发器可以看做JK触发器在J=K条件下的特例。

特性方程：

    Q^(n+1) = T非Q^n + TQ^n非
